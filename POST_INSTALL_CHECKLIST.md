# Post-Install Checklist

After `./install.sh` finishes, complete the following manual steps.

## Accounts and Apps

- Sign in to Google Chrome.
- Configure Logitech G Hub and grant Input Monitoring permission.
- Sign in to GitHub Desktop.
- Open Sublime Merge and apply your preferences.
- Sign in to WhatsApp.
- Open SF Symbols once to complete first-run setup.
- Open Cloudflare Warp and complete first-run setup.
- Open LocalSend and complete first-run setup.
- Open VLC and complete first-run setup.
- Open Raindrop.io and import data from iCloud.
- Open Hammerspoon and grant the requested Accessibility and Automation permissions.

## macOS Settings

- Set hot corner: System Settings -> Desktop & Dock -> Hot Corners -> bottom-left = Show Desktop.
- Set Dock Magnification to Large.
- Enable Tap to Click: System Settings -> Trackpad -> Point & Click.
- Adjust display resolution and arrangement as needed.
- Add your home directory to the Dock if desired.
- Add `cemozturk` or `Developer` to Finder Sidebar Favorites.
- Create additional Spaces for each display.

## Optional

- Import terminal profile from `settings/CMS.terminal` if needed.
